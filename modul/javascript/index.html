JavaScript

How to write

It can be written inside <head> or <body>, using <script>

Note:
*

Best practice is to put it at the end of <body>.

1. Better user experience browser parses the code file from top to bottom, therefore if the script is placed inside the <head>, the browser will get busy executing the script and won't be able to render the content of the page.

2. Code dependency one piece of code needs to read or use elements from another piece of code. So placing <script> at the end of <body> will ensure that all these elements are rendered by the browser.

**

There are exceptions when using third party codes, sometimes the code has to be placed inside <head>.
