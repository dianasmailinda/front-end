Block Element Modifier (BEM)

=> A popular naming convention

The concept is to think of webpages as bunch of blocks/components/moduls, these blocks can contain elements, and can be modified.


Naming conventions:

  • To seperate block and element: block__element

  • To seperate block and modifier: block--modifier


e.g:

We have 3 cards to define products, each card contains header and body. One card has differrent style compared to the others (Popular product).

Inside header we have a layout to display price.

Inside body we have lists of each product specifications and a button.


For case like this, here is how we gonna name our classes (check code)


Normal card:

card > (card__header > card__price) + (card__body > card__list + btn)


Popular card:

card card--popular > (card__header > card__price) + (card__body > card__list + btn)
