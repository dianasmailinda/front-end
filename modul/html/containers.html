Containers

  • Generic

    • div

      • Block-level element

      > Always starts on a new line and fill up the entire width of the page

      e.g.

      Lorem
      ipsum dolor sit amet, consectetur adipisicing elit. Beatae, id!

    • span

      • Inline element

      e.g.

      Lorem ipsum dolor sit amet, consectetur adipisicing elit. Beatae, id!

  • Semantic

    For same cases we use these containers instead of the generic containers to help search engine better understands the pages and what they contain

    • article

      • An independent, self-contained content.

      • Forum post
      • Comments
      • Reviews
      • Product cards

    • figure

      • An element for figures. It doesn't have any visual characteristics

      Figure element can optionally have a caption (<figcaption>) that can be above or below the figure.

    • mark

      • Used to highlight text

    • time

      • Used to represent a specific period of time

      It may include the datetime attribute to translate dates into machine-readable format.
