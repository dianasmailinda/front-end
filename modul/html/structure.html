Bulding blocks:

  • <header> is used to represent introductory content, which can belong to a page, section or article.
  • <main> is used to represent the main content of the page.
  • <footer> is used to represent introductory content, which can belong to a page, section or article.
  • <aside> is used as a side bar for advertising or other contents that is not directly related to the main content.
  • <section> is used to group related content.

Note

  • Every page can have only 1 main element

  • Every section should have a heading

  • <header> and <footer> can be used inside <section> or <article>
