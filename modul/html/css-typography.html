Flash of Unstyled Text:

How the texts appear while the web is downloading the embedded fonts.

font-display:

auto

: the browser decides which behavior will applied. its different from one browser to another.


swap

: the browser will use the fallback font then swap it with custom font when available. (-) user may experience a layout shift


fallback

: the browser will use the fallback font while downloading the custom font. if the custom font is not downloaded within that short period, the fallback font is gonna be used.


block

: the browser will hide the text until the custom font is downloaded.


optional

: just like fallback, but it will try to download the font in the background then store it in its cache, so in the next load browser will apply this custom font.
