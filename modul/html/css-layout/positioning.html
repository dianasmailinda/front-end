Absolute

Note:

!!To position an element absolutely, we must first position the container relatively


When we position an element absolutely, the element will be removed from the normal flow of the page. Means from the parent's POV, that element doesn't exist.

We can imagine as if its on a separate layer.

So now we got 2 layers.

1 for container, pink, and blue.

1 for yellow


That's why the blue replace the yellow position, because the container can only see blue and pink now.
