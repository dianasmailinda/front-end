px

The size will remain the same on any screen size

width: 300px

height: 300px


%

The size relative to its parent

width: 50% (parent's width: undefined)

height: 50% (parent's height: 600px)


Note:

  • If the width of the parent isn't set, it will set to 100% of the window size by default. Therefore, the width of the box above is dependent to the width of the window. (Resize the window to see)

  • If the height of the parent isn't set, a block level element height is 0 by default

viewport

The size relative to the viewport

width: 50vw

height: 50vh


em/rem

em: The size relative to the font size of the current element

rem: The size relative to the font size of the root element (html element = 16px)

font-size: 10px

width: 15em (15 x 10px)

height: 15rem (15 x 16px)
