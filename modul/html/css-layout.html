Positioning

Static

By default the position of all elements is static => not positioned, they appear exactly in their normal position


Relative

To position an element we set it to relative so we can position this element relative to its normal position


Absolute

Positioning an element relative to its container

Floating Elements


float: left

The element will float to the left side of the container and all the following elements will flow around it


float: right

The element will float to the right side of the container and all the following elements will flow around it


Note:

- Float can be tricky. Whenever we use it, we have to clear it after, so it wont damage the other elements.

- It is not recommended to use float for layout, since now we have FlexBox and Grid
